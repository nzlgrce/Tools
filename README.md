# 🛠️ Tools – All-in-One Utility Android App

**Tools** is a versatile all-in-one Android application that combines essential everyday utilities into a single, lightweight, and user-friendly platform. Designed for convenience and efficiency, the app eliminates the need to install multiple separate applications by providing a centralized toolkit for common mobile tasks.

## 📌 Overview

The **Tools App** integrates multiple device features and smart utilities into one seamless experience. Whether it's capturing moments, managing Bluetooth connections, recording audio, or performing quick calculations, this app serves as a compact digital toolbox for daily use.

Ideal for students, professionals, and general users who want quick access to essential tools in one place.

## ✨ Features

### 📷 Camera

* Capture photos directly within the app
* Simple and fast access to device camera functionality

### 🔵 Bluetooth

* Enable/disable Bluetooth
* Discover and connect to nearby devices

### 🎙️ Voice Recorder

* Record high-quality audio
* Save and manage recordings easily

### 🔊 Text-to-Speech (TTS)

* Convert text into spoken audio
* Useful for accessibility and multitasking

### 🎤 Speech-to-Text (STT)

* Convert voice input into text in real time
* Great for quick notes and hands-free typing

### ⏱️ Stopwatch

* Track time with precision
* Start, pause, and reset functionality

### ⏲️ Timer

* Set countdown timers 
* Alerts or notifications when time is up
* Ideal for productivity, workouts, or reminders

### 🔦 Flashlight

* Instantly turn your device into a flashlight
* Quick toggle for emergency or low-light use

### 🧮 Calculator

* Perform basic arithmetic operations
* Fast and responsive interface

### ➕ More Tools

* Additional utilities can be added and expanded over time

## ⚙️ Tech Stack

* **Platform:** Android (Java)
* **IDE:** Android Studio
* **APIs Used:**
  * Camera API
  * Bluetooth API
  * Speech Recognition & TTS APIs
  * Media Recorder
  * Device Hardware Controls
